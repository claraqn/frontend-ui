## 2021 캡스톤디자인3

[Light Blue React](https://flatlogic.com/templates/light-blue-react) 템플릿을 변형해서 사용했습니다 :)

<h3>사용한 템플릿 페이지</h3>

* Bootstrap 4+ & SCSS
* Responsive layout
* React Chart.js
* Simple login / logout
* Styled Bootstrap components like buttons, modals, etc

<hr>

<h3>프로젝트 실행방법</h3>

```
npm install
```

```
npm run
```
